# Ping Pong SDL3

A classic 2D Ping Pong game developed using C++ and the SDL3 library. 

## Description
This project is a simple implementation of the arcade classic Pong. It features a player-controlled paddle and an AI-controlled opponent.

## Features
* Player vs AI gameplay.
* AABB (Axis-Aligned Bounding Box) collision detection for the ball and paddles.
* Texture loading and alpha blending using the `stb_image` library.
* Smooth, framerate-independent movement using delta time.

## Controls
* **W** - Move Paddle Up
* **S** - Move Paddle Down
* **Space** - Launch the Ball

## Requirements
* C++17 or higher.
* Visual Studio 2022 (recommended, solution files are included).
* SDL3 library.
